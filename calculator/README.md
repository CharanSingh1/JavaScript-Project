# Calculator Using HTML, CSS & JavaScript

A simple and responsive calculator built with HTML, CSS, and JavaScript. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an intuitive user interface.

### Features

* Basic arithmetic calculations
* Responsive design
* Clean and modern UI
* Interactive button functionality

### Technologies Used

* HTML
* CSS
* JavaScript

This project is ideal for beginners looking to practice DOM manipulation, event handling, and JavaScript fundamentals.
